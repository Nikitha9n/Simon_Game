# Simon Says Game 🎮

A classic memory game built with HTML, CSS, and JavaScript. 

### [Click here for Live Demo](YOUR_GITHUB_PAGES_LINK_HERE)

## How to Play
1. Press any key on your keyboard to start.
2. The game will flash a color. Remember it!
3. Click the buttons in the correct sequence.
4. Each level adds a new color to the sequence.
5. If you click the wrong color, it's Game Over!

## Technologies Used
* HTML5
* CSS3
* JavaScript (Vanilla)
